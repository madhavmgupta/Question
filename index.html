Instructions on how to complete
the Multiple Choice Questions during an examination

Note: There are variations to the instructions per module exam which are not included in these general instructions

  1. Answers to all questions are to be recorded on the multiple-choice form, either in the format A-E or True/False.
  2. Mark one box only to indicate the answer you consider correct for each question.
  3. Fill up your personal details carefully.
  4. All questions are compulsory.
  5. You are allowed to submit only once, make sure that you have correctly the questions before submission.
  6. Make sure you clicked on finish button to successfully complete the test.
  7. Form will be active for half an hours. For session-I 10.05AM to 10.35PM and for session-II 12.05PM to 12:35PM.
